# API публичной библиотеки электронных книг

```
Список книг с пагинацией http://127.0.0.1:8000/books_list/
Список авторов с пагинацией http://127.0.0.1:8000/authors_list/
Фильтрация книг по автору http://127.0.0.1:8000/books_list/?author=%Автор%20Имя0
Фильтрация книг по названию http://127.0.0.1:8000/books_list/?name=Название книги 3
Фильтрация книг по кол-ву страниц:
    http://127.0.0.1:8000/books_list/?page=%<5
    http://127.0.0.1:8000/books_list/?page=%>5
    http://127.0.0.1:8000/books_list/?page=5
Фильтрация авторов по имени http://127.0.0.1:8000/authors_list/?name=Автор%20Имя0
```

## Установка
```
pip install -r requirements.txt
```
